## Project Overview
This repository contains the network topology models and routing configurations built in Cisco Packet Tracer for the Computer Networks practical module assignment.

## Included Files
1. **Router Basic Configuration-Task 1.pkt** - A 1-router using study the basic router configuration.
2. **Static Routing Configuration-Task 2.pkt** - A 3-router triangular network layout implementing manual static routes to establish connectivity across local networks.
3. **Dynamic Router Configuration-Task 3.pkt** - A linear 3-router topology using RIPv2 classless dynamic routing protocol.And the same linear 3-router topology updated to use EIGRP (Autonomous System 10) dynamic routing.

## Network Configurations Summary
* **Subnets Used:** * Task 2 LANs: 192.168.1.0/24, 192.168.2.0/24, 192.168.3.0/24
  * Task 3 LANs: 172.16.0.0/16, 173.16.0.0/16
* **Verification:** All topologies successfully converge and pass standard end-to-end ICMP ping connectivity tests.
